# 低代码试题生成

## 题目

1、根据用户输入知识点、生成题目类型、生成数量进行题目生成
2、读取知识库官网文档进行出题
3、选择题使用迭代组件给每个题目标题加上括号（训练一下迭代组件）
4、最后输出有选项和答案

使用到的组件节点
1、开始
2、知识检索
3、条件分支
4、LLM
5、参数提取器
6、迭代
7、代码执行
8、结束
